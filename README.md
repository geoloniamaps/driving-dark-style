# driving-dark-style

自動車走行時に見やすい地図スタイル・ダークモード (Driving-friendly map style - Dark mode)

ダーク背景に高コントラストの道路表示で、運転中にちらっと見るだけで道路情報が把握できるスタイルです。夜間走行にも適しています。

## プレビュー

https://geoloniamaps.github.io/driving-dark-style/

## 使い方

```
https://geoloniamaps.github.io/driving-dark-style/style.json
```

## 開発

```bash
npm install
npm start
```

## ビルド

```bash
npm run build
```

`docs/style.json` が生成されます。GitHub Pages で配信されます。
