# driving-style

自動車走行時に見やすい地図スタイル (Driving-friendly map style)

ダーク背景に高コントラストの道路表示で、運転中にちらっと見るだけで道路情報が把握できるスタイルです。

## プレビュー

https://geoloniamaps.github.io/driving-style/

## 使い方

```
https://geoloniamaps.github.io/driving-style/style.json
```

## 開発

```bash
npm install
npm start
```

## ビルド

```bash
npm run build
```

`docs/style.json` が生成されます。GitHub Pages で配信されます。
